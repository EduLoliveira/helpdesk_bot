# Aqui estamos gerenciando um projeto onde "simulamos" um bot;
# Realizando as seguintes interações.
-- Assim realiza a função de:
    - Monitoramento de chamados.
    - Notificações automáticas.
    - Geração de relatórios, analise de dados.
    - Integração com sistemas de chatBot (API).
    - Abertura de chamados, e confirmação automatizada.


-- Para configurar o ambiente de desenvolvimento, siga os passos abaixo:
    - Python -m venv venv # Para criar um ambiente virtual chamado venv.

-- Ativamos o ambiente virtual (venv):
    - venv\Scripts\activate  #No Windows 
    - source venv/bin/activate #No macOS/Linux  
-- Para atualizar todos pacotes disponiveis PIP, assim que criar e ativar ambiente venv:
    - pip install --upgrade pip 

-- Instalamos as dependências listadas no arquivo requirements.txt:
    - usando o comando: pip install -r requirements.txt.
